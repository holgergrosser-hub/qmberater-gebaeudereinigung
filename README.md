SEO-optimierte Landing Page für die ISO 9001 Zertifizierung in der Gebäudereinigung.

## Dateien

```
index.html                                    – Landing Page
qmberater-logo.svg                            – QMBerater Logo
holger-grosser-qm-guru-web-1024x977__4_.webp  – Portrait
netlify.toml                                  – Netlify Konfiguration
```

## Deployment auf Netlify

### Option 1: Über GitHub
1. Repository auf GitHub erstellen (z.B. `qmberater-gebaeudereinigung`)
2. Alle Dateien hochladen
3. In Netlify: "Add new site" → "Import an existing project" → GitHub Repo auswählen
4. Build settings werden automatisch aus `netlify.toml` übernommen
5. Deploy!

### Option 2: Manuell
1. In Netlify: "Add new site" → "Deploy manually"
2. Gesamten Ordner per Drag & Drop hochladen

## Custom Domain

In Netlify unter "Domain settings":
- Subdomain: `qmberater.info/iso-9001-gebaeudereinigung/`
- Oder als Subdomain: `gebaeudereinigung.qmberater.info`
